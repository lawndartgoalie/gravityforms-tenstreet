Enables gravity forms to transfer form inputs to Tenstreet database via API.

== Description ==

Integrates gravity forms with Tenstreet api. Transfer first name, last name, email, phone number, city and state.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `gravity-forms-tenstreet.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Settings -> Forms -> Tenstreet Settings and fill out the fields with your tenstreet credentials
1. On each form there is a Tenstreet settings page to enable/disable sending api data
1. On each notification setting form there is an option to enable/disable sending api results in the email

== Frequently Asked Questions ==

= How do I get my Tenstreet credentials =

Contact your Tenstreet sales rep or send an email to integrations@tenstreet.com.
